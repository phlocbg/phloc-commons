Idea og phloc-settings

There are 3 different elements within this project. These are:

  • Configuration
  • Preferences
  • Settings

These different types are outlined as follows:

Configuration

A configuration is a read-only set of application properties. They are provided from the outside (e.g. from a system administrator) and cannot be changed while the application is running.

If one or more configuration elements should be overrideable with user settings preferences are the way to do it.
